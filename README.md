# Random Task Generator

**Автор:** Карпеченкова Арина

## Описание
Приложение генерирует случайные задачи для учёбы, спорта или работы.  
Пользователь может:
- Сгенерировать случайную задачу из предопределённого списка
- Добавить свою задачу с выбором типа
- Фильтровать историю по типу задачи
- Сохранять и загружать историю в JSON

## Требования
- Python 3.7+
- Стандартные библиотеки: `tkinter`, `random`, `json`, `os`
