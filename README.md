# Weather Diary — Дневник погоды

**Автор:** Карпеченкова Арина Леонидовна

## Описание программы

GUI-приложение для ведения дневника погоды. Позволяет:
- Добавлять записи (дата, температура, описание, осадки)
- Просматривать все записи в таблице
- Фильтровать по дате и температуре (> заданного значения)
- Сохранять записи в JSON-файл и загружать из него
- Удалять выбранные записи

## Как запустить

1. Убедитесь, что установлен Python 3.6+
2. Скачайте файл `weather_diary.py`
3. Запустите в терминале/командной строке:
   ```bash
   python weather_diary.py
